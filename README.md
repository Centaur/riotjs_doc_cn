对应 riotjs版本 2.2.4

## 前言

[Riot](https://muut.com/riotjs/) 小巧，无侵入性，只辅助不强制，学习成本超低，兼容到IE8。基本上我在使用React时所有的痛点在Riot中全部没有。强烈推荐。
